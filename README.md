Зависимости.
```bash
pip install -r requirements.txt
```

Миграции
```bash
python manage.py makemigrations
python manage.py migrate
```

Очистка базы данных
```bash
python manage.py flush
```

Создание пользователя
```bash
python manage.py createsuperuser
```





Запуск сервера
```bash
python manage.py runserver
```
